# Atividade Pratica 3

- Projeto livre, porém deve ter complexidade o suficiente (discutir ideias com o professor)
- Ideias:
    - Outras métricas de distorção/qualidade para codificação de vídeo (SSD, SATD, MSE, PSNR)_
    - Preditor intra quadros para blocos 16x16 do padrão de codificação de vídeo H.264/AVC
    - Módulo de convolução (para filtros ou CNNs)
    - ULA com operações extras, como multiplicação, divisão, raiz quadrada, etc.
        - Não repetir com outra equipe_
- Se alguma das ideias acima for escolhida, pedir material extra ao professor
- O que entregar:
    - VHDL e arquivos de projeto do Quartus (*.qpf, *.qsf)
    - Testbenches (pelo menos do toplevel, idealmente para cada módulo descrito)
    - Relatório em formato de artigo IEEE (entre 2 e 4 páginas)
    - Apresentação (slides)
- Avaliação da entrega + apresentação para a turma
